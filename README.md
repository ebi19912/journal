
Project Overview: The Journal-Categorizer is a dedicated script built to streamline the process of organizing academic literature. It automates the search and classification of journals, allowing researchers to manage large volumes of scholarly publications with ease.

Technical Features:

Automated Categorization (jurnal.py): The core engine that processes journal data and applies classification logic to group similar publications.

Data Management: Handles raw input data (such as 333.txt) and transforms it into organized academic records.

Scalability: Designed to handle diverse sets of journals across different scientific domains.

Technical Stack:

Language: Python (100%).

Capabilities: Text parsing, Metadata extraction, and Data organization.
